# 👨‍💻 Guilherme Feitosa Cunha

**`Desenvolvedor FullStack`**

Com foco em desenvolvimento full stack, atuo com React no front-end e Node.js no back-end, utilizando os frameworks Express e NestJS. Tenho domínio de TypeScript e experiência na criação de aplicações completas, desde a modelagem de dados até o deploy. Trabalho com bancos de dados relacionais e não-relacionais, utilizando ORMs como Sequelize, TypeORM e Prisma. Tenho experiência com integração de APIs REST, versionamento com Git (GitHub e GitLab), testes com Jest e Testing Library, e conhecimentos em Docker. Já realizei deploys em ambientes como Vercel, Railway e AWS Lightsail, além de configurar servidores com Apache e HTTPS para projetos pessoais. Possuo familiaridade com o ambiente Linux (Ubuntu), onde gerencio uma instância na AWS para fins de estudo. No front-end, desenvolvo interfaces responsivas e acessíveis, sempre priorizando código limpo, organização e escalabilidade. Atuo com autonomia ou em equipe, buscando constante evolução técnica e entregas de alto impacto.
